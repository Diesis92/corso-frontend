La direttiva IF produce un output anziché un altro 
IF testa un valore, se è true produc ein outpot quanto contenuto all'interno delle graffe altrimenti nulla
se sarà il costruto else viene scritto dopo la parentesi if che rappresenta 
 dice altrimenti produci questo in output 
 questo va scritto nel file mixing 
 @mixin font-sizer($ size, $max-size:24px) {
    @if ($size <=$max-size) {
        font-sizer:$size;
    }
    @else {
        font-size:$max-size;
    }
 }

$list: p, h1, h3

    }
}
